Repository Title: Nonlinear Causality and Strong Hyperbolicity of Einstein-IS Theories from DNMR

This repository is a supplemental repository for the preprint of the paper titled

"Nonlinear Causality and Strong Hyperbolicity of Einstein-Israel-Stewart Theories of Transient Relativistic Fluid Dynamics"
by Ian Cordeiro, Enrico Speranza, F'{a}bio S. Bemfica, Marcelo Disconzi, and Jorge Noronha

All calculations were written by Ian Cordeiro (itc2@illinois.edu). 


"IS from DNMR no diffusion.nb" is the main document consisting of the general calculation of the full Israel-Stewart (IS) theory from Denicol-Niemi-Molnar-Rischke (DNMR) theory with bulk and shear contributions.

PLEASE REFER TO THIS DOCUMENT FIRST TO UNDERSTAND NOTATION/CONTEXT.


There are currently two supplementary notebooks

"Generalized Maxwell-Cattaneo.nb" contains all relevant calculations for the generalized (second-order in transport coefficients) Maxwell-Cattaneo considered in the main paper.

"Conformal IS from DNMR.nb" contains all relevant calculations for the conformal limit of the IS system from DNMR without diffusion.
